   3. [ ] Dataproc

4. パートナーインターコネクトを利用する際に課金される対象と期間は？
   1. [ ] 1年単位で回路
   2. [ ] 1か月単位で回路
   3. [x] 1か月単位でVLAN
   4. [ ] 1時間単位で回路
   5. [ ] 1年単位でVLAN
   6. [ ] 1時間単位でVLAN

   - Dedicated Interconnectは、1時間当たり基本料金(VLAN)と1か月当たりの回路
